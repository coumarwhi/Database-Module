2. Explain when a database would be used. Explain when a text file would be used.

A: A database would be used to store, create, and manipulate data without using up memory. A text fie would be used for storing something that you want to be able to read directly, and that doesn't take up too much memory. 

3. Describe one difference between SQL and other programming languages.

A: SQL is declarative compared to other programming languages, which are procedural. 

4. In your own words, explain how the pieces of a database system fit together at a high level.

A: Databases display data using a table structure. With the use of SQL (a declarative language), you can get results from data, update data, and create new structures to store data. Written queries tell the database what you are looking for and the database engine finds the most efficient way to return the data you've requested. 

5. Explain the meaning of table, row, column, and value.

A: The columns define what the data represent, the rows make up a complete conceptual unit, value is the information within each cell. All of these make up a table.
